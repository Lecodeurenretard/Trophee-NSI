# Installer les dépendances nécessaires
1. Installer Python sur [python.org](https://www.python.org/downloads/). Téléchargez le _"install manager"_ et suivez les étapes.
2. Ouvrez la console en appuyant sur "Windows" puis en tapant `cmd`.
3. Installez Pygame en tapant `python -m pip install pygame-ce`, si ça ne marche pas remplacez `python` par `python3` ou `python3.14`.

Tout est prêt.
