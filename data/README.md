# Le dossier [data/](.)
Ici sont toutes les données qui ne sont pas du code source.

Les données textuelles (stats, ennemis, items, ...) utilisées par les classes comme `MonstreJSON` sont dans [JSON/](JSON/).

La distinction entre [sfx/](sfx/) (Sound EffeCTS) et [musique/](musique/) est faite car Pygame les traite différement.
